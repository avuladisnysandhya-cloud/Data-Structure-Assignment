## Programs in this Repository
1. **Q1 – Reverse a String using Stack**  
   - File: `q1_reverse_string_stack.c`  
   - Description: Takes a string as input and reverses it using a character stack.

2. **Q2 – Balanced Parentheses using Stack**  
   - File: `q2_balanced_parentheses.c`  
   - Description: Checks whether the parentheses in a given expression are balanced using a stack.

3. **Q3 – Next Greater Element using Stack**  
   - File: `q3_next_greater_element.c`  
   - Description: For each element in an array, finds the next greater element on the right using a stack.

4. **Q4 – Printer Queue Simulation**  
   - File: `q4_printer_queue.c`  
   - Description: Simulates a simple printer queue with options to add a document, print a document, and display pending documents.

5. **Q5 – Circular Queue Basic Operations**  
   - File: `q5_circular_queue.c`  
   - Description: Implements a circular queue using an array with enqueue, dequeue, peek, and display operations (menu-driven).

## How to Compile and Run (Linux / Mac / Windows with GCC)
```bash
# Compile
gcc q1_reverse_string_stack.c -o q1
gcc q2_balanced_parentheses.c -o q2
gcc q3_next_greater_element.c -o q3
gcc q4_printer_queue.c -o q4
gcc q5_circular_queue.c -o q5

# Run
./q1
./q2
./q3
./q4
./q5
```
